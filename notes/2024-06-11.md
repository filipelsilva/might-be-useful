# About ZFS

Command to create a pool in ZFS with RaidZ1 and two disks:

```bash
sudo zpool create -O compression=lz4 -O mountpoint=none -O xattr=sa -O acltype=posixacl -O atime=off -O recordsize=1M -o ashift=12 -m /mnt/data data raidz /dev/disk/by-id/ata-ST8000VN004-3CP101_WRQ01QF2 /dev/disk/by-id/ata-ST8000VN004-3CP101_WWZ3T73R
```

Commands to see and update the Seagate drives sleep patterns:
```bash
sudo openSeaChest_PowerControl -d /dev/sdb --idle_a 100 --idle_b 120000 --idle_c 600000 --standby_z 900000
sudo openSeaChest_PowerControl -d /dev/sdb --showEPCSettings
```
